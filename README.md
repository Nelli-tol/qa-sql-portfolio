# QA SQL Portfolio

This repository contains SQL query examples used by QA engineers
for data validation, integrity checks, and basic analytics.

The focus is on practical SQL tasks commonly performed during
backend and database testing.

---

## Scope

- Data validation and sanity checks
- NULL and duplicate detection
- Referential integrity (orphan records)
- Date and numeric range validation
- Aggregation and reconciliation checks
- Simple anomaly detection

---

## Tools & Stack

- SQL (ANSI SQL)
- PostgreSQL / MySQL compatible syntax
- Relational data models
- QA-oriented data analysis

---

## Project Structure

- `queries/` — SQL queries for QA data validation
- `datasets/` — sample datasets (CSV, demo data)
- `docs/` — notes and explanations

---

## Notes

This repository is intended for demonstration and learning purposes.
Queries are written in a QA mindset, focusing on detecting data issues
rather than building production analytics.
